$ linux-setup
GitHub Manifest Updater

One command for the boring machine setup.

Install and update the small scripts that make a fresh shell feel like home: resume picker, AI start menu, and future setup modules.

curl -fsSL https://setup.lost.plus/install.sh | bash
GitHub Pages backed Version-aware updates Local drift writes .new

Install

Bootstrap `~/.local/bin/linux-setup`, then install modules with `linux-setup install resume ai-menu`.

Update

`linux-setup update` compares in-file versions against remote files and pulls newer scripts in one pass.

Protect

If a managed file was edited locally, updates write a sibling `.new` file instead of clobbering your changes.
